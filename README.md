This is a simple library to help you using common concept in Rial Digital projects.

This library is using simple classes for generating RAPSA path format.

Also this project is going to use WASM for generating RSA keypair and sign data.

I use [rapsa-js](https://github.com/Pourbahrami/rapsa-js) for RAPSA generator, Thanks to [Hossein](https://github.com/Pourbahrami).
